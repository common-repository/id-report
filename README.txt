Exporte endereços completos dos pedidos concluídos no Woocommerce/IgnitionDeck para arquivo PDF e agilize o envio de suas encomendas.
 
== Description ==
 
Este plugin permite a exportação de informações de endereço e envio de pedidos feitos em campanhas Crowdfunding criadas utilizando o plugin IgnitionDeck jutamente com Wocoomerce. 

O plugin formata o endereço de envio deixando-o ideal para envio das encomendas, veja o Exemplo:

JOSÉ ALMEIDA
Rua Antonio Basílio , 1111 , casa Morro Branco
Natal - RN
CEP: 999-999

O usuário deverá selecionar: 

	* Um projeto IgnitionDeck
	* Em seguinda selecionar os pedidos que quer exportar
	* Clicar em "Gerar PDF"
	* 
	* Ou, selecionar os pedidos já carregados
	* do Woocommerce e clicar em "Gerar PDF"
